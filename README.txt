مهارت‌ها و تجربیات فنی:

معماری و الگوها: تسلط بر MVVM و Clean Architecture برای طراحی و توسعه اپلیکیشن‌های قابل نگهداری و مقیاس‌پذیر.

مدیریت وضعیت و جریان داده: تجربه کامل در استفاده از BLoC، Flow و Route برای مدیریت وضعیت اپلیکیشن و مسیریابی.

پایگاه داده: پیاده‌سازی و مدیریت داده‌ها با Room و SQLite برای ذخیره‌سازی محلی اطلاعات.

فریم‌ورک‌ها و زبان‌ها: توسعه با Jetpack Compose، Flutter، Java XML و Kotlin.

ابزارها و عملکرد: استفاده از Profiler برای بررسی عملکرد اپلیکیشن و Cursor برای کد سریع.

تست و تضمین کیفیت: تجربه در Android Testing برای اطمینان از عملکرد صحیح و پایدار اپلیکیشن‌ها.

این مجموعه تجربیات نشان‌دهنده توانایی شما در طراحی معماری مناسب، مدیریت وضعیت، پیاده‌سازی رابط کاربری پیچیده، بهینه‌سازی عملکرد و تضمین کیفیت اپلیکیشن‌های اندروید و فلاتر است.

کار با API: تجربه کامل در توسعه و مصرف APIها با Retrofit برای ارتباط اپلیکیشن با سرور و مدیریت داده‌های شبکه.
