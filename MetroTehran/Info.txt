Tehran Metro Navigation Application Project

Designed and developed an Android application using Java and XML to provide a comprehensive urban navigation experience.

Identified and displayed the nearest metro station based on the user’s current location.

Showed metro stations within a 2,500-meter radius, along with relevant distance and station details.

Provided key facilities and services available at each metro station to better inform users.

Displayed the complete metro map with precise station locations integrated into the map view.

Implemented full route navigation from the user’s location to the selected metro station.

Presented information about related bus and taxi lines connected to each metro station.

This project delivers a practical and comprehensive solution for urban commuters.
